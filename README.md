# GRIP-DFFI

GitHub may not directly render some large `.ipynb` notebook files correctly.

## How to View Notebook Outputs

For each experiment folder, open the file ending with:

`_OUTPUT_VIEW.md`

Those Markdown files show the notebook code and outputs directly in GitHub.

The `.ipynb` files are kept as the original source notebooks.
